# AI Course Source Template (Upload Input)

Use this structure for DOCX/PPTX/PDF source files uploaded to course framework drafts.

## Required Sequence
1. Course framework draft + upload + extraction must complete first.
2. Topic analysis/generation/finalize occurs next.
3. Lesson artifacts are created only after framework completion/finalization.

## Heading Blueprint
1. `Course Title`
2. `Course Description`
3. `Overview`
4. `Lesson 1: <Topic>`
5. `Lesson 2: <Topic>`
6. `Lesson 3: <Topic>`
7. `Key Takeaways`

## Authoring Guidance
- Keep each lesson section semantically distinct.
- Include enough body text per lesson for extraction and assignment quality.
- Keep heading hierarchy clean and consistent.
- Avoid placing key lesson content only in images.
