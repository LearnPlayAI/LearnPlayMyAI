# LearnPlay Demo Data Structure (Full Datamap)

Generated from `/antigravity/Cloud-On-Prem/shared/schema.ts` on 2026-03-23.

## Scope Decisions

- This document inventories **all 185 tables** and their columns.
- It differentiates **System Data** vs **Transactional Data**.
- It marks demo policy per table: required, seed/reference, optional/derived, or explicitly excluded.
- Excluded areas (as approved): packaging/entitlements, usage-limit licensing enforcement, AI draft/translation ops, guest/public sessions, platform cost/scheduled reporting ops.

## Integrity Rules (Non-Negotiable)

- Every generated record set must be tagged with a batch marker (`demo_batch_id`) and demo source flag.
- All FK relationships must resolve and stay tenant-safe (including onprem cross-org sharing rules).
- Timeline/state/version consistency must hold across enrollment, progress, quiz, purchase, refund, payout, and reporting views.
- Credit ledgers and financial aggregates must reconcile to source transactions.
- Generator actions must be backend-gated to cloud/onprem DEV+ACC only (never PRD).
